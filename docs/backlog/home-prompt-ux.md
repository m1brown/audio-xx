# Home Prompt UX Concept

Goal:
Improve the Audio XX home page prompt to help users ask useful questions without forcing them into a form.

## Primary Prompt

Describe your system, what you're hearing, or what you're curious about.

## Starter Prompts

- Assess my system and suggest upgrade directions
- Help me understand my listening preferences
- Help me find the right [category]

## Rotating Category

The last word rotates every few seconds to show different gear categories.

Possible categories:
- headphones
- speakers
- DAC
- amplifier
- IEMs
- streamer
- turntable
- speaker cables

Example rotation:
Help me find the right headphones
Help me find the right speakers
Help me find the right DAC

## UX Behavior

When the user enters a system description, detect components and show:

Detected components:
- [component list]

Button:
Save as system

## Design Principle

Conversation first.
Structure later.

Users should be able to ask questions naturally without filling out forms.
